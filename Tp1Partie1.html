XML Shema

Définition

XML Shema publié comme recommendation par le W3C en mai 2001 est un langage de description de format de document XML permettant de definir la structure et le type de contenu d`un document XML . Cette définition permet de vérifier la validité de ce document .

Modele de XML Schema

La recommendation spécifie la validation des document XML a partir d`un modele abstrait , elle en fournit de format XML .Pour ce modele abstrait, un shéma est un ensemble de composants

Exemple 

 
< ?xml version= "1.0" encoding ="UTF-8"?>
<xs :schema xmlns : xs = "https://www.w3.org/2001XMLSchema">
<xs:element name="personne">
    <xs:complexType>
	   <xs:sequence>
	     <xs:element name="nom"type "xs:string "/>
         <xs:element name="prenom"type "xs:string "/>
		 <xs:element name="date_naissance"type "xs:date "/>
		 <xs:element name="etablissement"type "xs:string "/>
		 <xs:element name="num_tel"type "xs:string "/>

	    </xs:sequence>
	</xs:complexType>	
</xs:element>
</xs:schema >
